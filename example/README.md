# Example Audio Files

This directory is for example audio files to test AudioCleaner.

## Usage

Place your test audio files here and run:

```bash
cd ..
./audio_cleaner.command
```

Then drag and drop files from this directory.

## Recommended Test Files

- **Clean speech**: To test baseline performance
- **Noisy speech**: Background noise, static
- **Low quality**: Phone recordings, compressed audio
- **Various formats**: WAV, MP3, FLAC

## Note

Audio files in this directory are ignored by git (except this README).
Add your own test files as needed.
